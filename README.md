#### *Prerequisites*

To setup the development environment for the project you will need the following:
````
Node.js npm and Metamask
````
#### *Installing:*

#### *Clone the repository*
```sh
git clone https://github.com/ZentaChain/ZentaSwap
````

install required packages:
````sh
$ npm install
````
run with:
````sh
$ npm start
````
